# ProjetoCompAplicada.CSharp

API desenvolvida em **C# e .NET 9**, utilizando o **Entity Framework Core** e o **MySQL**, com o objetivo de realizar consultas, listagens e filtros sobre os dados do sistema principal do projeto de Computação Aplicada.

---

## Objetivo

Esta aplicação foi criada como uma **API paralela de consulta e apoio**, conectada ao mesmo banco de dados utilizado pelo backend Java.  
Seu foco está em **operações de leitura (GET)**, com **paginação**, **filtros**, **organização de dados** e **boas práticas** de desenvolvimento em C#.

A ideia é demonstrar o uso do .NET como alternativa tecnológica no mesmo domínio do sistema, **sem substituir** ou **replicar** o backend original em Java.

---

## Tecnologias

- **Linguagem:** C# (.NET 9)  
- **ORM:** Entity Framework Core (Pomelo MySQL)  
- **Banco de dados:** MySQL  
- **Documentação:** Swagger  
- **Gerenciamento de ambiente:** `.env` e `appsettings.json`
